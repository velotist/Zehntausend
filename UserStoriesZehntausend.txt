User Stories für Chicago – Iteration #1

User Story Vorlage:
Als <Rolle/Anwendertyp> möchte ich <Ziel/Wunsch/Funktion>, um <Nutzen>

Titel: Spieler anlegen
Story:
Als Benutzer möchte ich mehrere Spieler anlegen, um gegeneinander spielen zu können
Akzeptanzkriterien:
	* zwischen zwei und vier Spieler
Status:
	* Erledigt

Titel: Namen anlegen
Story:
Als Benutzer möchte ich Spielern Namen vergeben, um sie auseinanderhalten zu können
Akzeptanzkriterien:
	* nur Vornamen
	* bis zu 50 Zeichen
Status:
	* Erledigt

Titel: Eine Rangliste anlegen
Story:
Als Benutzer möchte ich eine Rangliste mit den gewonnenen Spielen haben, um den Besten ermitteln zu können
Akzeptanzkriterien:
	* Name und Punkte
Status:
	* Offen

Titel: Spiel mehrmals spielen
Story:
Als Benutzer möchte ich das Spiel mehrmals spielen können, um eine Gewinnliste anlegen zu können
Akzeptanzkriterien:
	* Gewinnliste mit Spielernamen und deren Punkten
	* Wiederholung des Spieles
Status:
	* Offen

Titel: Spielstand speichern
Story:
Als Benutzer möchte ich vor Beendigung des Spieles den Spielstand speichern, um an diesem
Punkt weiterspielen zu können.
Akzeptanzkriterien:
	* Name, Id und Punkte sollen gespeichert werden
	* als Dateiformat soll csv verwendet werden
Status:
	* Erledigt
